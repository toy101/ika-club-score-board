# ika-club-score-board

## テストデータ

### 起動方法

`SEED_TEST_DATA` 環境変数でシード挙動を切り替える。

| 値 | 挙動 |
|---|---|
| 未設定 / `if-empty`（デフォルト） | `leagues` テーブルが空のときだけシードする |
| `force` | 既存のテストリーグを削除して強制的に再投入する |
| `off` | シードしない |

**DB が存在しない（または空の）場合 — デフォルトで自動シード：**

```sh
cd api
make run
```

**既存 DB にテストデータを上書き投入したい場合：**

```sh
cd api
SEED_TEST_DATA=force make run
```

シードされるデータ：テストリーグ（固定 ID `00000000-0000-4000-a000-000000000001`）・4 チーム・各チーム 4 名。

---

### テストデータの確認方法

#### API で確認

```sh
# リーグ一覧
curl http://localhost:8080/leagues

# テストリーグのチーム・メンバー一覧
curl http://localhost:8080/leagues/00000000-0000-4000-a000-000000000001/teams
```

#### UI で確認

WebUI（`pnpm dev`）が起動している状態でブラウザから開く：

```
http://localhost:3000/leagues/00000000-0000-4000-a000-000000000001
```

リーグ詳細ページで 4 チームとそれぞれのメンバー名が表示されていれば OK。
